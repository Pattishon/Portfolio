I used box-shadow to achieve this cool looking 'cards' effect.

  • I applied the dark, thin shadow, to imitate the actual shadow of the first page
  • Below I placed a shadow in a color of the front page, no blur effect on this one, to imitate the secod 'card.
  • For the second 'card' I applied the same style, darker shadow as for the first one.
  • And then repeat the process one more time. And that's it!
box-shadow: 2px 2px 3px 1px #aaa, 
8px 10px 0 1px #eee, 
8px 12px 3px 3px #aaa,
14px 20px 0 3px #eee, 
14px 22px 3px 5px #aaa;
